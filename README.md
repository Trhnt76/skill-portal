# Skill Portal

A personal engineering skill tracker and learning platform.
Built on Azure, deployed via GitHub Actions, defined in Bicep.

## Live Demo
https://YOUR-URL.azurestaticapps.net

## Stack
- Frontend: HTML/CSS/JS (React in Phase 2)
- Hosting: Azure Static Web Apps
- IaC: Bicep
- CI/CD: GitHub Actions
